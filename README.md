Installation process of project on your machine:
1. git clone - https://github.com/yashmalbhage/Predixion-AI-ASSI
2. Open the folder
  Frontend -
        1.cd taskmanagement_client
        2.npm i
        3.npm start
 
  Backend - 
        1. cd backend
        2.pip install -r requirements.txt
        3.uvicorn app.main:app --reload
